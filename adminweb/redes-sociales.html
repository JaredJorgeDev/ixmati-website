Proceso mensual

Cómo funciona

El servicio está planteado como una forma de trabajo mensual. El objetivo es producir contenido útil, revisarlo con tiempo y publicarlo sin improvisaciones de último minuto.

Recolección de información del negocio

Se recibe material, promociones, horarios, cambios del negocio y cualquier novedad del ciclo.

Definición de tono y objetivos

Se ajusta el estilo de comunicación según tipo de negocio, público y prioridad del mes.

Planeación de contenido mensual

Se arma calendario con publicaciones, stories y piezas promocionales según el plan.

Diseño y redacción

Ixmati prepara visuales, copies y estructura final de cada entrega mensual.

Revisión del cliente

El cliente valida el bloque del mes dentro del número de revisiones contratado.

Programación o publicación

Con aprobación final, el contenido queda calendarizado para salir en fecha y forma.

Las revisiones y ajustes incluidos se resuelven dentro del ciclo mensual acordado. Si el cliente solicita cambios fuera de esa ventana, podrán pasar al siguiente bloque de trabajo o cotizarse por separado.
Publicidad

Publicidad (Meta Ads y Google Ads)

La pauta publicitaria es un servicio separado del manejo de redes. Puede complementar el trabajo, pero no forma parte automática de los planes mensuales de contenido.

Lo que sí debe quedar claro

  • La publicidad pagada no está incluida en los planes de redes sociales
  • El presupuesto de anuncios siempre es cubierto directamente por el cliente
  • Ixmati no absorbe inversión publicitaria
  • Las campañas se trabajan como complemento cuando el negocio lo necesita

Regla operativa

El presupuesto de campañas nunca está incluido en los planes. Es una inversión directa del cliente hacia la plataforma publicitaria.

Ixmati no mezcla presupuesto publicitario con honorarios de servicio. La inversión en anuncios es independiente.
Inputs del cliente

Lo que el cliente debe entregar para operar correctamente

Mientras más claro y puntual sea el material base, más estable, útil y personalizado puede ser el resultado mensual.

Material base

  • Accesos a redes sociales
  • Logo
  • Fotos del negocio, productos o servicios
  • Promociones vigentes
  • Horarios y datos correctos
  • Links de redes
  • Referencias visuales si existen
  • Aprobación oportuna del contenido
  • Material con derechos de uso

Condición operativa

Si la aprobación del bloque mensual se retrasa, también se mueve la programación. Este servicio depende de una ventana clara de revisión para sostener orden y continuidad.

Si el cliente no entrega material suficiente, Ixmati podrá trabajar con stock, plantillas o contenido base, lo cual puede limitar la personalización del resultado.

Operación interna

Cómo se opera este servicio

Se vende como un trabajo mensual ordenado. No está planteado para reaccionar diario a cualquier solicitud, sino para sostener presencia con contenido ordenado.

Ciclos mensuales

Se trabaja por ciclos mensuales definidos, con fechas de entrega, revisión y salida de contenido.

Planeación previa

El contenido se planea; no se improvisa diario ni se arma sobre la marcha sin orden.

Aprobación del cliente

Se requiere aprobación del cliente para publicar. Sin aprobación, el calendario también se ajusta.

No es atención instantánea

El servicio no funciona como atención bajo demanda ni como respuesta inmediata a cualquier cambio diario.

Logística mensual

Cómo se maneja la logística del mes

El trabajo mensual se organiza por ciclos. No se trabaja improvisando publicaciones aisladas todos los días.

Inicio de ciclo

Se recopila información, promociones, prioridades y cambios relevantes para el siguiente mes o periodo activo.

Producción en bloque

Las piezas se diseñan y redactan como paquete para mantener coherencia visual y acelerar revisiones.

Revisión controlada

Se hacen los ajustes dentro del número de revisiones del plan. Cambios extra o fuera de tiempo pueden cotizarse aparte.

Publicación calendarizada

Una vez aprobado, el contenido se agenda para que el perfil mantenga ritmo estable durante el mes.

Integración con web

Redes sociales funcionan mejor cuando tienen a dónde llevar la atención.

La red social ayuda a generar visibilidad, recordación y movimiento. El sistema web es lo que convierte esa atención en una acción concreta del cliente.

Las redes sociales atraen atención. El sistema web convierte esa atención en pedidos, citas o contacto real.

Cómo se integra con los planes web

  • Tier 1: no incluye campañas
  • Tier 2: incluye configuración de Google Ads, pero no incluye presupuesto
  • Tier 3: incluye configuración de Google Ads + Meta Ads, pero no incluye presupuesto

Relación entre web y campañas

Los planes web incluyen la preparación de campañas (estructura, copies, configuración), pero la inversión publicitaria corre por cuenta del cliente.

La configuración técnica y la inversión publicitaria son cosas distintas. Ixmati prepara la base; el cliente fondea la campaña.
Redes sociales atraen atención

Mueven alcance, muestran actividad, presentan promociones, recuerdan horarios y mantienen al negocio visible.

Stories, posts y promociones atraen miradas y generan interés.

Cierre

Ixmati no maneja redes para subir por subir.

Las gestiona como parte de un sistema digital que ayuda a que el negocio se vea activo, confiable y listo para vender. Cuando redes y web trabajan juntas, la presencia deja de ser decorativa y empieza a servir al negocio.

Trabajamos con pocos negocios por zona para mantener calidad y control del contenido.
